PART 3: TECHNICAL & INTERACTION GUIDELINES
Framework: React + Tailwind CSS.

Components: Use Shadcn UI for base components (Buttons, Cards, Accordion, Sheet for mobile menu).

Icons: Use lucide-react.

Responsiveness:

On Mobile (<768px): The "Floating Hero" images should stack into a hidden or simplified grid, prioritizing the H1 and CTA.

Motion & UX (Performance Focused):

Library: motion/react (do not use Framer Motion standard package if heavy, use the lightweight version if possible, or framer-motion with lazy loading).

Implementation:

Wrap the Hero images in <motion.div> with a floating animation: animate={{ y: [0, -10, 0] }} transition={{ duration: 4, repeat: Infinity, ease: "easeInOut" }}.

Scroll Reveal: Use initial={{ opacity: 0, y: 30 }} and whileInView={{ opacity: 1, y: 0 }} for sections.

Interactions:

Buttons: whileHover={{ scale: 1.05 }}.

Cards: whileHover={{ y: -5 }}.
